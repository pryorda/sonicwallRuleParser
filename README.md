# Sonicwall Rule Parser [Python 3 Update]
Parses Sonicwall Rules, Groups, and Services from a settings export file.

# Example Usage
python ./parser.py sampleExport.exp
```
==========================================================
================== Firewall Rules ========================
==========================================================

LAN,WAN,Any,Blaze Meter Group,HTTP,Allow,ExampleRule

==========================================================
```
